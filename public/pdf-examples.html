Example 3: Math Formula Usage in Questions

Use LaTeX syntax in your question text:

Inline Math:

The quadratic formula is $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$

Display Math:

Find the integral:
$$\int_{0}^{\pi} \sin(x) dx$$

Common Math Symbols:

Greek letters: $\alpha, \beta, \gamma, \Delta, \Omega$
Operators: $\pm, \times, \div, \neq, \leq, \geq$
Calculus: $\int, \sum, \prod, \lim, \partial$
Sets: $\in, \notin, \subset, \cup, \cap$
Arrows: $\rightarrow, \Rightarrow, \leftrightarrow$

Example 6: Preview Before Generating

POST /api/admin/pdf/preview 
{
  "questions": [1, 2, 3, 4, 5],
  "template": "magazine",
  "language": "both",
  "options": {
    "column_count": 2,
    "render_math": true,
    "show_solutions": true
  }
}

Returns HTML preview that you can display in browser before generating the final PDF.

Configuration Options Reference

Option Type Description
column_count integer (1-4) Number of columns in magazine layout
column_gap string Gap between columns (e.g., "8mm", "10mm")
render_math boolean Enable mathematical formula rendering
math_engine string "katex" or "mathjax"
font_size integer (8-20) Base font size in pixels
page_padding string Page margins (e.g., "15mm")
